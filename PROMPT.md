# Steve's Shop - GitHub Pages Website Build Prompt

## Overview

Build a beautiful, single-page GitHub Pages website for **Steve's Shop** — a handcrafted wood cutting board and charcuterie board business. This site will serve as a showcase/mock storefront and future stepping stone toward an Etsy shop. It should be deployable via GitHub Pages (static HTML/CSS/JS only, no backend).

## About the Business

Steve is a passionate woodworker who hand-crafts cutting boards, charcuterie boards, and serving boards from carefully sourced hardwoods. He loves the **hunt for the perfect raw wood** — finding unique slabs, rescuing interesting pieces, and sourcing exotic hardwoods. For Steve, the journey of finding the right piece of wood is just as rewarding as transforming it into a functional work of art. Each piece is one-of-a-kind. Every board is branded with his signature circular logo stamp.

**Price range:** $140–$165 per board

## Design Direction

- **Warm, artisan, and earthy** — think craft market meets modern web design
- Use a color palette inspired by the wood tones in the photos: rich walnut browns, warm honey/amber, creamy maple, and deep purpleheart accents
- Typography should feel handcrafted but legible — a serif or slab-serif for headings, clean sans-serif for body
- The photos are the stars — make them large, prominent, and well-presented
- Should feel premium but approachable — this is a craftsman who puts love into every piece, not a mass-production operation
- Fully responsive (mobile-first) — buyers will likely share the link and view on phones
- Smooth scroll navigation
- Subtle animations on scroll (fade-in for product cards, parallax on hero if appropriate)

### 4. "The Wood" / Materials Section
- A section dedicated to the **types of wood** Steve works with — this is key to his passion and a major selling point
- Create a visual grid or showcase of wood types with:
  - **Walnut** — Rich, dark chocolate brown with flowing grain. The backbone of the collection.
  - **Maple** — Creamy white with subtle figure. Creates beautiful contrast.
  - **Cherry** — Warm pinkish-red that deepens with age. A timeless classic.
  - **Purpleheart** — Vibrant purple that stands out in any combination. Sourced from Central/South America.
  - **Yellowheart** — Bright golden yellow. Adds warmth and pop.
  - **Mahogany** — Rich reddish-brown with distinctive ribbon grain.
  - **Wenge** — Dark brown/black with dramatic coarse grain texture.
  - **Padauk** — Vivid orange-red that ages to a deep burgundy.
  - **Lacewood** — Features a unique lace-like spotted pattern. A rare find.
  - **Spalted Maple** — Maple transformed by fungi into a one-of-a-kind canvas of dark lines and organic patterns.
- You can use color swatches or cropped details from the product photos to illustrate each wood type
- Include a brief note about Steve's sourcing philosophy — the hunt, the relationships with local sawmills, finding fallen trees, etc.

## Technical Requirements

- **Static site** — HTML, CSS, and vanilla JS only (must work on GitHub Pages)
- Use the actual product photos from the `/photos` directory — reference them as `photos/IMG_XXXX.jpeg`
- Optimize for performance: use lazy loading for images, consider CSS for responsive image sizing
- Responsive grid layout (CSS Grid or Flexbox)
- Smooth scroll behavior
- Mobile hamburger menu
- SEO meta tags (title, description, Open Graph tags for social sharing)
- Add structured data (JSON-LD) for local business / product schema — this will help when transitioning to Etsy and for Google visibility
- Favicon (can be a simple wood-grain themed icon or the letter S)
