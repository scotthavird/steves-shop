# Steve's Woodcraft

Handcrafted cutting boards and charcuterie boards made from carefully sourced exotic hardwoods.

**Live site:** [scotthavird.github.io/steves-shop](https://scotthavird.github.io/steves-shop/)

## About

Steve is a passionate woodworker who hand-crafts one-of-a-kind boards from walnut, maple, cherry, purpleheart, yellowheart, mahogany, wenge, padauk, lacewood, and spalted maple. Every piece starts with the hunt for the perfect slab and ends with a food-safe finish and Steve's signature branded logo.

**Price range:** $140–$165

## Product Lines

- **Live-Edge Charcuterie Boards** — Single-slab boards with natural bark edge
- **Multi-Wood Cutting Boards** — Exotic hardwoods laminated in stripe patterns
- **Carving Boards with Juice Groove** — Live-edge with routed juice channel, some with hand-forged iron handles
- **Spalted & Figured Wood Boards** — Naturally patterned spalted maple showcasing nature's artwork

## Tech Stack

Static single-page site built for GitHub Pages:
- HTML / CSS / vanilla JS
- No build tools or dependencies
- Optimized images (~250-400KB each, down from 4-6MB originals)
- SEO meta tags, Open Graph, and JSON-LD structured data
- Responsive design with mobile navigation
